# vareal-ai-team

Vareal株式会社 AI社員チーム紹介ページ

## Deploy

```bash
git pull origin main
docker compose up -d --build
```

## Access

- Internal: http://localhost:18900
- Public: https://ai-team.vareal.group (via Cloudflare Tunnel)

## Stack

- Nginx (Alpine)
- Docker Compose
- Cloudflare Zero Trust Tunnel
